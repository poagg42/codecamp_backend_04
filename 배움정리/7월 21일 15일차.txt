정규화 
중복 데이터를 분리하는 것

테이블들을 작은 테이블로 나눈다.

나눠진 결과물을 정규형이라고 부른다

중복 데이터에 초점을 맞추자

정규형(Normal Form) 종류

1NF 2NF 3NF BCNF 4NF 5NF

보통 1NF 2NF 3NF << 실무용

1NF

원자값이 아닌 값 제거
하나의 컬럼에는 데이터가 하나씩 있어야 한다.

하나의 컬럼에 데이터가 두 개씩 들어있는 것을 -> 다가속성 이라고 한다.

1NF에서 하나의 컬럼에 여러 속성이 들어있는 것을 -> 복합속성

복합 속성 제거해야된다 ex) A001 레고 1개 -> 

상품번호 / 상품명 / 상품수량
  A001  /  레고  /   1개

분리한다는 것은 다시 합칠 수 있어야 한다

다시 합칠 수 있는 방향을 생각해서 컬럼을 만들어야 된다.

주문번호 / 날짜 / 이름 / 나이

주문번호 / 상품번호 / 상품명 / 상품수량

각각을 구분할  수 있는 핵심 키 

이런 경우

주문번호 / 상품번호

P00O / A0001

P0001/ A0022

2NF

상품명이 주문번호와 어떤 관련이 있을까

서로 관련없는 데이터들이 모여있을 때

이럴 때도 

[주문](테이블) [주문][상품번호/상품명] [상품] 테이블
